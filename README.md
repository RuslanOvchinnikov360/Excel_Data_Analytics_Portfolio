# Excel Data Analytics Portfolio

## Overview
This repository contains a collection of Excel-based analytics projects focused on business reporting, KPI development, sales analysis, dashboard creation, and data transformation.
The projects demonstrate practical Excel skills commonly used in Data Analyst and Business Analyst roles, including advanced formulas, lookup functions, conditional calculations, date analysis, and data visualization.
